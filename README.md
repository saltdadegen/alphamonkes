# AlphaMonkes Website

A modern, responsive website built with HTML, CSS, and JavaScript.

## Setup

1. Clone the repository
```bash
git clone https://github.com/saltdadegen/alphamonkes.git
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm start
```

## Structure

- `/styles` - CSS styles and components
- `/images` - Image assets
- `/js` - JavaScript files

## Development

- Use `npm start` to run the development server
- Changes will automatically reload in the browser

## Deployment

The site can be deployed using GitHub Pages or other hosting services.
